# Eleganckie CV Karen Tonoyan - Instrukcja Użytkownika

## O Stronie

**Cel:** Profesjonalna strona CV prezentująca doświadczenie zawodowe, umiejętności i wykształcenie Karen Tonoyan.

**Dostęp:** Publiczny - każdy może odwiedzić stronę bez logowania.

---

## Powered by Manus

Ta strona CV została zbudowana przy użyciu nowoczesnego stosu technologicznego:

- **Frontend:** React 19 + TypeScript + Tailwind CSS 4 + shadcn/ui
- **Responsywność:** Pełna obsługa mobile, tablet i desktop
- **Wdrożenie:** Auto-scaling infrastruktura z globalnym CDN
- **Wydajność:** Szybkie ładowanie i płynna nawigacja

---

## Korzystanie ze Strony

### Przeglądanie CV

Strona wyświetla pełne CV Karen Tonoyan zorganizowane w sekcje:

1. **Kliknij "CV"** w górnym prawym rogu, aby pobrać lub wydrukować CV w formacie PDF.
2. **Przewijaj stronę** aby zobaczyć wszystkie sekcje: Wykształcenie, Doświadczenie Zawodowe, Języki, Umiejętności i Zainteresowania.
3. **Informacje kontaktowe** znajdują się w lewym panelu bocznym - telefon, email i adres.

### Pobieranie CV

Kliknij przycisk "CV" w nagłówku strony. Możesz:
- Wydrukować stronę (Ctrl+P lub Cmd+P)
- Zapisać jako PDF za pomocą opcji drukowania przeglądarki
- Pobrać plik CV w formacie PDF

### List Motywacyjny

Przycisk "List Motywacyjny" jest dostępny w nagłówku - kliknij go, aby uzyskać dostęp do listu motywacyjnego (funkcja dostępna po dodaniu zawartości).

## Zarządzanie Stroną

### Panel Zarządzania

Aby zmienić zawartość CV lub wygląd strony, użyj panelu zarządzania:

- **Settings (Ustawienia):** Zmień nazwę strony, logo lub widoczność
- **Dashboard:** Monitoruj statystyki odwiedzin i wydajność
- **Database:** Zarządzaj danymi (jeśli strona zostanie rozszerzona o bazę danych)
- **Secrets:** Zarządzaj kluczami API i zmiennymi środowiskowymi

### Edycja Zawartości CV

Aby edytować zawartość CV:

1. Otwórz plik `client/src/pages/Home.tsx` w edytorze kodu
2. Zaktualizuj dane w sekcjach (Wykształcenie, Doświadczenie, Umiejętności, itp.)
3. Zapisz plik - strona automatycznie się odświeży
4. Sprawdź zmiany w przeglądarce

### Dostosowanie Wyglądu

Aby zmienić kolory lub styl:

1. Edytuj plik `client/src/index.css` - zmień zmienne CSS
2. Aktualne kolory: ciemne tło z złotymi akcentami
3. Zmień wartości `oklch()` aby dostosować paletę kolorów
4. Zmiany pojawią się natychmiast na stronie
